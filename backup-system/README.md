# Automated Backup & Recovery System

A Bash-based backup automation tool that creates compressed backups with timestamps.

## Features

- Automated backup creation
- Timestamped backup files
- Gzip compression
- Tar archiving
- Linux shell scripting

## Technologies

- Bash
- tar
- gzip
- Linux

## Usage

```bash
chmod +x backup.sh
./backup.sh
```

## Example Output

```text
backups/backup_2026-06-01_11-30-15.tar.gz
```
